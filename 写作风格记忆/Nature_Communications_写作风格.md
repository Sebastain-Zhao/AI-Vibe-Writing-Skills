# Nature Communications 写作风格记忆

> 基于论文《Mitigating stimulated Brillouin scattering in multimode fibers with focused output via wavefront shaping》分析

## 📌 核心特点

### 整体风格
- **期刊类型**: Nature Communications (综合性科学期刊)
- **风格特点**: 简洁明了、逻辑清晰、重点突出、图文结合
- **结构**: 标准IMRaD结构 (Introduction, Methods, Results, and Discussion)

---

## 📝 各部分写作要点

### 1. 标题 (Title)

**结构公式**:
```
[动词-ing形式] + [核心问题] + with + [方法/手段]
```

**示例**:
> "Mitigating stimulated Brillouin scattering in multimode fibers with focused output via wavefront shaping"

**要点**:
- 动词-ing形式开头 (Mitigating, Enhancing, Controlling...)
- 直接点明研究内容
- 副标题说明创新方法
- 简洁有力，避免冗长

---

### 2. 摘要 (Abstract)

**结构**: 单段落，200-250词

**三段式逻辑**:
1. **背景** (1-2句): 研究问题的重要性
2. **方法** (1-2句): 创新方法简述
3. **结果** (2-3句): 主要发现和意义

**关键句式**:
```
✓ "Here we demonstrate..."     → 引出主要发现
✓ "We show that..."             → 阐述机制
✓ "Our results reveal..."       → 揭示结果
✓ "These findings..."           → 总结意义
```

**示例开头**:
> "Stimulated Brillouin scattering (SBS) is a major limitation for high-power fiber lasers. Here we demonstrate a method to mitigate SBS in multimode fibers using wavefront shaping..."

---

### 3. 引言 (Introduction)

**漏斗式结构**:

```
第1段: 宏观背景
    ↓
第2-3段: 文献综述 (成就+局限)
    ↓
第4段: 研究动机 (新方法/技术)
    ↓
第5段: 本文贡献
```

**引用特点**:
- 格式: `[n]` 数字编号
- 密度: 每段3-5个引用
- 格式: `Author et al. [n]`
- 范围: `[1-3]`, `[4,5]`

**关键句式**:
```
背景引入:
- "...has attracted significant attention due to..."
- "...is of great importance for..."

文献综述:
- "Previous studies have demonstrated... [n]"
- "However, these methods suffer from..."
- "Recently, ... has emerged as a promising approach [n]"

研究动机:
- "Despite these advances, ... remains challenging"
- "To address this issue, ..."

本文贡献:
- "In this work, we present..."
- "Our approach offers several advantages:..."
```

---

### 4. 方法 (Methods)

**组织结构**:
```
Methods
├── Experimental setup
├── Sample preparation
├── Numerical simulations
└── Data analysis
```

**写作要点**:
- 使用小标题分隔
- 详细参数描述
- 引用示意图
- 被动+主动语态混合

**关键句式**:
```
装置描述:
- "The experimental setup consists of..."
- "A schematic of the setup is shown in Fig. 1."
- "We use a [device] to [purpose]..."

参数说明:
- "The fiber has a core diameter of..."
- "The laser operates at a wavelength of..."
- "The pulse duration is..."

模拟方法:
- "Numerical simulations are performed using..."
- "The model is based on..."
```

---

### 5. 结果 (Results)

**呈现方式**:
- 每个主要结果对应一个Figure
- 小标题分隔不同实验
- 定性描述 + 定量数据

**关键句式**:
```
图表引用:
- "Figure 1 shows..."
- "As shown in Fig. 2a,..."
- "(see Fig. 3 and Supplementary Fig. 1)"

观察描述:
- "We observe that..."
- "The results demonstrate..."
- "The data reveal..."

强调发现:
- "Interestingly, ..."      → 意外发现
- "Notably, ..."             → 重要发现
- "Surprisingly, ..."        → 出乎意料
- "As expected, ..."         → 符合预期
- "Consistent with..., ..."  → 与...一致
```

---

### 6. 讨论 (Discussion)

**内容要点**:
1. 机理解释
2. 与已有研究对比
3. 优势与局限
4. 应用前景

**关键句式**:
```
机理解释:
- "This can be understood by considering..."
- "The underlying mechanism is..."
- "These observations can be explained by..."

对比分析:
- "Compared to previous work [n], our method..."
- "In contrast to conventional approaches,..."
- "Unlike [previous method], our approach..."

优势讨论:
- "The advantage of this approach is..."
- "Our method offers several benefits:..."
- "This technique enables..."

局限性与展望:
- "One limitation of this study is..."
- "Future work will focus on..."
- "These findings open avenues for..."
```

## 🎯 语言特点

### 时态使用

| 时态 | 使用场景 | 示例 |
|------|----------|------|
| **一般现在时** | 普遍真理、研究结果 | "The results show that..." |
| **一般过去时** | 实验过程、观察现象 | "We observed a peak at..." |
| **现在完成时** | 引用前人工作 | "Previous studies have shown..." |

### 语态选择

**主动语态为主** (增强可读性):
```
✓ "We demonstrate..."
✓ "We show..."
✓ "We find..."
✓ "We observe..."
```

**被动语态用于**:
- 实验设置描述
- 通用方法说明
- 强调对象而非执行者

### 过渡词

```
递进:     Furthermore, Moreover, In addition, Additionally, Also
转折:     However, Nevertheless, Although, While, Yet, In contrast
因果:     Therefore, Thus, Consequently, As a result, Hence
举例:     For example, For instance, Specifically, In particular
总结:     In summary, Overall, Taken together, Collectively
对比:     Compared to, In contrast, Unlike, Similarly
```

## 📊 引用风格

### 文中引用

| 类型 | 格式 | 示例 |
|------|------|------|
| 单个 | `[n]` | `[1]` |
| 多个 | `[n,m]` | `[2,3]` |
| 范围 | `[n-m]` | `[4-6]` |
| 带作者 | `Author et al. [n]` | `Smith et al. [7]` |

### 参考文献格式

```
[编号] 作者. 标题. 期刊名 卷, 页码 (年份).

示例:
[1] Agrawal, G. in Nonlinear Fiber Optics, 5th edn 
    (ed. Agrawal, G.) 353-396 (Academic Press, 2013).
[2] Stolen, R., Bjorkholm, J. & Ashkin, A. Phase-matched 
    three-wave mixing in silica fiber optical waveguides. 
    Appl. Phys. Lett. 24, 308-310 (1974).
```

```

```

```
"We present a method for..."
"Our approach is based on..."
"We combine... with... to..."
```

### 结果陈述
```
"The results demonstrate that..."
"We achieve... under... conditions"
"The maximum/minimum... is..."
"An improvement of... is observed."
```

### 结论总结
```
"In summary, we have demonstrated..."
"Our findings reveal that..."
"This work provides a new approach to..."
"These results open new possibilities for..."
```

## 🎓 写作技巧总结

### DO's ✅
- 使用主动语态增强可读性
- 每段一个中心思想
- 图表与文字相互补充
- 客观陈述，避免过度推测
- 使用具体数据支持论点
- 保持术语一致性

### DON'Ts ❌
- 避免冗长复杂的句子
- 避免使用第一人称单数 (I)
- 避免模糊表述 (some, many, etc.)
- 避免在结果部分过度解释
- 避免引用过多导致阅读困难
