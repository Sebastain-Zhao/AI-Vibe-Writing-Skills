## ❌ 中文写作禁忌

### 1. 避免被动句式
- ❌ 被...所...
- ❌ 由...所...
- ✅ 使用主动语态

**实际验证**：6篇中文文献中 **0例** 被动句

### 2. 减少机械过渡词
实际使用情况：
- **因此**（9次）- 建议减少
- **此外**（1次）- 建议减少
- **另外**（1次）- 建议减少

推荐替代：
- 使用逻辑连接：实验结果表明... / 基于上述分析...
- 使用关键词承接：这一现象说明... / 上述方法实现了...
- 直接陈述：无需过渡词

### 3. 避免AI味词汇
实际文献中 **未发现** 以下词汇（说明真实文献质量高）：
- 深入探讨 / 错综复杂 / 见证
- 多维度的 / 助力 / 赋能
- 底层逻辑 / 颗粒度

### 4. 避免翻译腔
- ❌ 对于...来说
- ❌ 在...方面
- ❌ 进行...的研究（改为：研究...）
- ❌ 实现...的优化（改为：优化...）

### 5. 避免绝对化结论
- ❌ 证明了 / 确认了 / 毫无疑问
- ✅ 表明了 / 提示 / 可能

## 🎯 实用写作建议

### 中文论文写作流程

1. **开头段落**
   - 直接陈述研究背景和动机
   - 避免"随着...的发展"等套话
   - 用具体数据说明问题重要性

2. **方法描述**
   - 使用主动语态：我们提出/设计/实现...
   - 或使用：本文提出/本研究设计...
   - 避免被动：被提出/被设计...

3. **结果陈述**
   - 使用hedging：结果表明/实验显示/数据提示
   - 注入具体数值：提高了X dB / 降低了Y %
   - 避免绝对化：证明了/确认了

4. **讨论分析**
   - 用逻辑连接替代过渡词
   - 基于数据进行分析
   - 可以推测/可能的原因是...

5. **结论部分**
   - 总结核心贡献
   - 使用hedging表达
   - 指出未来方向

### 高频句式模板

**提出方法**：
- 我们提出了一种新的...方法
- 本文设计并实现了...
- 本研究构建了...

**描述结果**：
- 实验结果表明，所提方法...
- 数据显示，系统性能提高了X dB
- 结果提示，该方法可以有效...

**对比分析**：
- 与传统方法相比，本方法...
- 相比于文献[X]，我们的方法...
- 实验验证了所提方法的有效性
