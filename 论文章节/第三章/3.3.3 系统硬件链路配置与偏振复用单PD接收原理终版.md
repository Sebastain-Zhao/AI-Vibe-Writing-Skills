基于前文对ODB抗色散机理的分析，本节将阐述双偏振光双二进制（DP-ODB）系统的硬件实现方案与接收原理。所提方案的核心设计思想在于将系统复杂度集中于ONU侧，而保持OLT侧的简洁性，以确保与现有PON架构的兼容性和经济可行性。
DP-ODB系统的发射端硬件链路如图3.7所示，主要由任意波形发生器（Arbitrary Waveform Generator, AWG）、电放大器、双驱动马赫-曾德尔调制器（DD-MZM）及偏振复用模块构成。信号生成流程如下：首先，两路独立的伪随机二进制序列（PRBS）数据经双二进制预编码与部分响应成型处理，生成三电平电信号。AWG以92 GSa/s的采样率输出幅度为176 mV的基带信号，经23 dB增益的电放大器放大后，驱动带宽为25 GHz的DD-MZM。调制器偏置在null点，实现光双二进制调制。为构建双偏振信号，调制后的光输出经50:50光纤耦合器分为两路：X偏振支路延迟20个符号以实现两路数据的去相关；Y偏振支路通过可调光衰减器引入3 dB功率衰减，使两路光信号形成严格的2:1功率比。随后，两路信号分别经偏振控制器（Polarization Controller, PC）对准水平偏振X与垂直偏振Y，由偏振合束器（Polarization Beam Combiner, PBC）合并为单一光纤输出。该偏振复用结构虽增加了发射端复杂度，但为代价换来了接收端仅需单PD即可实现直接检测，显著降低了ONU侧的硬件成本与复杂度。

DP-ODB系统的接收端采用极简架构，仅由掺铒光纤放大器（EDFA，Erbium-Doped Fiber Amplifier）前置放大器、35 GHz带宽的PIN光电探测器及高速采样示波器组成，如图3.7所示。偏振复用单PD接收的核心原理在于利用光电探测器的偏振不敏感特性，实现对两路正交偏振信号的标量叠加检测。设X偏振支路的ODB信号光功率为$P_x(t)$，Y偏振支路的光功率为$P_y(t)$，由于Y支路经过3 dB衰减，两路功率满足$P_x(t) = 2P_y(t)$。经偏振合束器合并后，总光信号可表示为：
	\[\vec{E}(t)={{\vec{e}}_{x}}{{E}_{x}}(t)+{{\vec{e}}_{y}}{{E}_{y}}(t)\]
其中$\vec{e}_x$和$\vec{e}_y$分别为水平与垂直方向的单位偏振矢量，$E_x(t)$和$E_y(t)$为对应的光场幅度。在光电探测器处，由于PD的平方律检波特性仅响应光强（即光场的模平方），且对偏振态不敏感，因此输出光电流与两路偏振分量的光强之和成正比：
	\[I(t)=R\cdot \left[ |{{E}_{x}}(t){{|}^{2}}+|{{E}_{y}}(t){{|}^{2}} \right]=R\cdot \left[ {{P}_{x}}(t)+{{P}_{y}}(t) \right]\]
其中$R$为探测器的响应度。将$P_y(t) = 0.5 P_x(t)$代入上式，可得：
	\[I(t)=R\cdot \left[ {{P}_{x}}(t)+0.5{{P}_{{{x}'}}}(t) \right]\]
式中$P_x(t)$和$P_x'(t)$分别表示X偏振支路的两路独立ODB信号（经20符号延迟去相关）。两路非相干叠加后形成四电平强度信号，电平间隔呈现1:1:1的均匀分布特征。这种非相干叠加检测机制的优势在于完全避免了传统相干接收所需的90°光混频器和平衡探测器阵列，显著简化了接收机结构。同时，由于两路偏振信号在光域保持正交，在电域通过标量叠加自然实现4电平幅度调制，无需额外的偏振解复用操作。
经ADC采样后的数字信号进入DSP模块进行后续处理。DP-ODB系统的DSP流程相对简洁，主要包括以下步骤：（1）重采样与同步：将示波器采集的原始数据重采样至符号率的整数倍，并进行符号定时恢复；（2）均衡处理：采用FFE和DFE级联结或者加入Volterra均衡构补偿信道损伤。
